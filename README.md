# one-button-platformer

what up gang


this is platformer that uses only one key to jump!

temu geometry dash except the player character doesn't flip and only gets squashed a bit

there are 4 different themes so far depending on ur aesthetic


press space, tap, or click to jump. there are 8 levels, and they get progressively more difficult. after finishing each level you checkpoint. if you win the game and want to prestige or something, click on a different theme.

spikes and drops are insta-kill.




made using js, css, and html. it's my first time separating the files instead of having it all on one doc! :3
